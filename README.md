# Student Productivity Dashboard (Excel)

## Overview

This project analyzes how different factors such as study hours, sleep quality, and stress levels affect student productivity. The analysis is performed using Microsoft Excel with pivot tables and charts to create an interactive dashboard.

---

## Objectives

* Understand the relationship between study hours and productivity
* Analyze the impact of sleep quality and stress levels
* Compare productivity under different conditions
* Present insights through a clear and visual dashboard

---

## Dashboard Features

* KPI cards showing distribution of:

  * High Productivity
  * Low Productivity
  * Moderate Productivity
* Pivot table-based analysis
* Comparative charts for:

  * Good Sleep & Low Stress
  * Poor Sleep & High Stress
* Clean and structured layout for easy interpretation

---

## Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data grouping and percentage analysis

---

## Dataset

The dataset includes student-related attributes such as:

* Study hours
* Sleep quality
* Stress levels
* Productivity levels

**Note:**
The dataset file name contains "mental health", but this project specifically focuses on productivity analysis using those factors.

---

## Key Insights

* Study hours alone do not strongly determine productivity
* Sleep quality and stress levels have a noticeable impact
* Students with good sleep and low stress show more balanced productivity
* Poor sleep and high stress conditions show higher variability in productivity

---

## Files in Repository

* `student_productivity_dashboard.xlsx` → Final Excel dashboard
* `student_mental_health_burnout.csv` → Dataset used for analysis

---

## Conclusion

This project demonstrates how Excel can be used as a powerful tool for data analysis and dashboard creation. It highlights the importance of combining multiple factors rather than relying on a single variable like study hours to understand productivity.
